# Flight Performance Analysis Using Hadoop and Hive

This project analyzes millions of U.S. airline performance records using a scalable big‑data pipeline built with Hadoop and Hive. The goal is to identify delay patterns across airports and carriers and support data-driven operational insights.

## Key Features
- Built Hive tables for flights, airports, and carriers
- Analytical queries to compute delay metrics
- Identification of top delay-heavy airports and carriers

## Results
- Clear patterns in arrival vs. departure delays
- Insights into operational bottlenecks in the aviation system

## Skills Demonstrated
HiveQL, Hadoop, big‑data processing, SQL analytics, data quality validation
